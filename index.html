    Introduction

    Welcome to the Imdb Unofficial API! You can use our API to access Imdb content. In examples we are using symfony component called The HttpClient Component

    Authentication

    To authorize, use this code:

    <?php
$api_key = YOURAPIKEY;
$query_parameters = QUERYPARAMETERS;

$httpClient = HttpClient::create([
    'headers' => ['X-AUTH-API' => $api_key],
]);

$response = $httpClient->request('GET', 'https://admin.bunny-holding.com/api', ['query' => $query_parameters,]);
?>

    Make sure to replace YOURAPIKEY with your API key. Make sure to replace QUERYPARAMETERS with parameters as it is defined in API.

    Each project has different API key. Before making any request with your api key please be sure you configured allowed titles in admin

    API key must be included in all API requests to the server in a header that looks like the following:

    X-AUTH-API: YOURAPIKEY

    You must replace YOURAPIKEY with your API key from your project in [admin](https://admin.bunny-holding.com).

    Query Parameters

    Query parameters are used to generated specific response. You can combine it in any way you want.

    Extra

$query_parameters = [
    'extra' => [
        ['function' => 'popularTitles', 'parameters' => ['limit' => 10], 'properties' => ['imageSrc']],

    Item (title)

    <?php
$query_parameters = [
    'item' => [
        'id' => 587,
        'entity' => 'Title',
    ],
];
?>

    Required: false

    Parameter Required Allowed Values Format Description
    entity yes Title string static text, identify collection
    id yes integer the ID of major title
    properties no properties array array of properties
    relations no relations array array of relations
    methods no methods array array of methods

            ['function' => 'randomDescription'],
            ['function' => 'listSeasons', ],
            ['function' => 'listSeasonsWithEpisodes', ],

            ['function' => 'previousEpisode', 'properties' => ['season', 'episode']],

    ],
];

    otherTitles
    Name: limit
    Required: yes
    Type: int
    Max value: 10

    Valid Options: id, name, description, imageSrc, season, episode, duration, contentRating, metascore, locations, keywords, awards, rating, releasedYear, releasedMonth, releasedDay, episodeParent, slug, category

    		 other titles based on major title
    randomDescription None. None. generate random description
    listSeasons None. None. list of seasons if major entity is a tv show
    listSeasonsWithEpisodes None.
    Default: id, name, slug, imageSrc
    Valid Options: id, name, description, imageSrc, season, episode, duration, contentRating, metascore, locations, keywords, awards, rating, releasedYear, releasedMonth, releasedDay, episodeParent, slug, category

    		 list of seasons with episodes as objects
    nextEpisode None.
    Default: id, name, slug, imageSrc
    Valid Options: id, name, description, imageSrc, season, episode, duration, contentRating, metascore, locations, keywords, awards, rating, releasedYear, releasedMonth, releasedDay, episodeParent, slug, category

    		 next episode if major entity is episode
    previousEpisode None.
    Default: id, name, slug, imageSrc
    Valid Options: id, name, description, imageSrc, season, episode, duration, contentRating, metascore, locations, keywords, awards, rating, releasedYear, releasedMonth, releasedDay, episodeParent, slug, category

    		 previous episode if major entity is episode
